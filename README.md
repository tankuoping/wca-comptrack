# WCA Upcoming Competitions

Look up upcoming WCA competitions for any competitor by name or WCA ID.

## Features
- Search by name, partial name, or exact WCA ID
- Pick the correct person from search results
- Shows upcoming competitions with registered events (via WCIF)
- First event start time in competition's local timezone
- Links to WCA competition page, competitiongroups.com psych sheet
- Add to Google Calendar

## Deploy to Vercel

### Option A — Vercel CLI
```bash
npm install -g vercel
npm install
vercel
```

### Option B — GitHub + Vercel dashboard
1. Push to a GitHub repo
2. Go to vercel.com → New Project → Import repo
3. Vite is auto-detected → click Deploy

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:5173
